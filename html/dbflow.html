java에서 database 자료 가져오는 방법

필요한 자료 준비

java에서 database에 접속해서 자료를 가져오기 위해서는 다음의 자료가 필요하다.
  • 접속할 서버 주소
  • 접속할 서버에서 사용하는 포트 번호
  • DBMS의 데이터베이스 이름
  • DBMS의 사용자 이름 / 비밀번호

접속할 서버 주소의 형식

jdbc:postgrespl://127.0.0.1:43432/testdb

위와 같은 형식으로 적는데 순서대로,
  • DB와 연동하는데 jdbc를 이용(java니까)
  • postgresql을 이용
  • 주소는 127.0.0.1
  • 포트번호는 43432
  • 데이터베이스 이름은 testdb
라는 걸 알려준다.
